Задачи

MyDataSpace позволяет выполнять JS-скрипты (задачи) на сервере по расписанию — раз в час/сутки/неделю/месяц.

Задачи предназначены для выполнения рутинной работа с данными, например синхронизации. Они не предназначены для сложных вычислений.

{{ 'https://myda.space' | replace: "https://","https://" }}

Код задачи выполняется в песочнице на сервере точно также как клиентской код в браузере.

Благодаря этому задачи можно отлаживать непосредственно в браузере из административной панели.
