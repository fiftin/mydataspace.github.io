Бекенд — это серверная часть системы, предоставляющаю API для поиска, чтения, загрузки и изменения данных.

Админ панель

С помощью неё администратор создаёт скрипты, загружает файлы и данные, настраивает доступ к ним.

{{ 'https://myda.space' | replace: "https://","https://" }}

Приложения

Настройка доступа к данным осуществляется через приложения.

Чтобы разрешить сайту добавлять и изменять данные, необходимо создать приложение и указать в нем URL этого сайта.

В клиентском коде при подключению к бекенду необходимо использовать ключ этого приложения.

{{ 'https://myda.space' | replace: "https://","https://" }}

Задачи

MyDataSpace позволяет выполнять скрипты JavaScript (задачи) на сервере по расписанию — раз в час, сутки, неделю или месяц.

{{ 'https://myda.space' | replace: "https://","https://" }}

Код задачи выполняется в песочнице на сервере точно также как в браузере.

Благодаря этому задачи можно отлаживать непосредственно в браузере из административной панели.

{{ 'https://myda.space' | replace: "https://","https://" }}

Ресурсы

На MyDataSpace можно хранить файлы (ресурсы).

Для картинок создаются превьюшки нескольких размеров.

Загружать файлы можно как через административную панель, так и с помощью API.
