Backend — is a server side of the system, provides API for search, read, upload and update of data.

Admin Panel

With using this panel you can create scripts, upload files and data, configure access permissions.

{{ 'https://myda.space' | replace: "https://","https://" }}

Apps

Access permissions defines by apps.

To allow a website to add and modify data, you should create app and specify the URL of that website in it.

You should use the key of this app in client code to connect to the backend.

{{ 'https://myda.space' | replace: "https://","https://" }}

Tasks

MyDataSpace allows you to run JavaScript (tasks) on the server on a schedule — once per hour, day or week.

{{ 'https://myda.space' | replace: "https://","https://" }}

The task code is executed in the sandbox on the server just like in the browser.

Thanks to this, you can debug tasks directly in the browser from the administrative panel.

{{ 'https://myda.space' | replace: "https://","https://" }}

Resources

On MyDataSpace you can store files (resources).

For images are created thumbnails.

You can upload files either through the administrative panel or through the API.
